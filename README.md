## 🔍 Методология тестирования

- **Точки проверки**: более 55 локаций по всему миру (Европа, Азия, Северная и Южная Америка, Ближний Восток).
- **Объект проверки**: публичные тестовые эндпоинты каждого CDN (без указания конкретных доменов).
- **Измеряемый параметр**: полное время ответа (TTFB) в секундах, включая редиректы и ошибки.
- **Обработка данных**: для каждого региона взято минимальное значение из всех доступных тестов.
- **Критерий включения**: провайдеры, имеющие собственные серверы в Российской Федерации или официально работающие с российским трафиком.

## 🚀 Дальнейшие шаги

1. **Проверьте самостоятельно** – запросите тестовый доступ у двух-трёх лидеров для вашего региона.
2. **Протестируйте с реальными файлами** (изображения, JavaScript, видео) – задержка может отличаться от времени на пустой ответ.
3. **Сравните цены** и дополнительные опции (защита от DDoS-атак, техническая поддержка, соглашение об уровне услуг).
